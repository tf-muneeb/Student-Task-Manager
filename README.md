# 📚 Student Task Manager

A Flutter-based task management application designed for students, featuring REST API authentication, Firebase Firestore integration, and Provider-based state management.

---

## ✨ Features

🔐 User authentication via DummyJSON REST API
✅ Create, view, edit, and delete tasks
🔄 Real-time data synchronization with Firebase Firestore
📱 Clean, modern, and responsive UI
💾 Cloud-based persistent storage

---

## 🛠️ Tech Stack

| Category         | Technology         |
| ---------------- | ------------------ |
| Framework        | Flutter & Dart     |
| Database         | Firebase Firestore |
| API              | DummyJSON REST API |
| State Management | Provider           |
| Platforms        | Android, iOS, Web  |

---

## 📂 Project Structure

```
lib/
 ├── models/            # Data models (Task Model)
 ├── providers/         # State management (Task Provider)
 ├── screens/           # UI screens (Login, Home, Add/Edit/View Tasks)
 ├── services/          # API & Firebase services
 ├── widgets/           # Reusable UI components
 ├── firebase_options.dart
 └── main.dart
```

---

## 🚀 Getting Started

1. Clone the repository
2. Run `flutter pub get`
3. Configure Firebase credentials
4. Run `flutter run`
